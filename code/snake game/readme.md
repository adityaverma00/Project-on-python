# Snake Game
This is an snake game where i used turtle to create a interface to make a new screen instead of using a terminal
## how to play this
- press arrow up to go up
- press arrow down to go down
- press arrow left to go left
- press arrow right to go right
## info
- There are 4 files food.py, main.py, scoreboard.py, snake.py
- if the snake touches the wall or if the snake touches its own tails then its game over
- the more food you eat the more you grow and the more the challenge begin
